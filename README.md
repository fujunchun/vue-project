# 介绍

> 基于vue webpack模板 1.3.1，根据业务开发需要，搭建了一套通用模板

## 包含内容

### 基础

1. vue + vuex + vue-router全家桶
2. ui库：element-ui
3. 请求库：axios
4. 语言国际化：vue-i18n
4. 数据mock: koa + koa-router + mockjs

### demo演示

> 提供简单博客演示

1. 登录登出
2. 权限控制
3. 语言切换

## 使用 

1. 启动mock服务器
```
npm run mock
```

2. 启动项目
```
npm run start
```
